# NBA Fan Sentiment NLP Pipeline & Dashboard 🏀📊

## Project Overview
"Data isn’t just numbers in a spreadsheet—it’s human behavior." 

This project explores the relationship between live-game score realities and digital crowd sentiment during the New York Knicks' Game 4 turnaround on June 10, 2026. By engineering an end-to-end NLP data pipeline, this project maps out how a massive 29-point physical deficit correlates directly to real-time public emotional shifts.

## Tech Stack
* **Language:** Python (Jupyter Notebook)
* **Libraries:** Pandas, NumPy, vaderSentiment
* **Business Intelligence:** Tableau Desktop (Packaged Workbook)

## Core Architecture & Workflow
1. **Pipeline Validation Simulation:** Programmatically engineered a 500-node simulated text log to model social media commentary streams across distinct chronological game phases, bypassing API rate limits for immediate testing.
2. **VADER Sentiment Analysis:** Utilized Python to parse complex colloquial slang and social syntax, translating raw textual expressions into explicit numerical emotional scores ranging from -1.0 (despair) to +1.0 (hype).
3. **Advanced Dashboard UI/UX:** Built a dark-themed Tableau dashboard utilizing custom calculated fields to aggregate mood by game phase and a reversed-axis layout to map sentiment trends directly against physical score margins.
